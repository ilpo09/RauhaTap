# Valwen Brand Kit

## Core Idea

Valwen is a calm panic-attack aid, not a general meditation app. The product should feel immediate, quiet, trustworthy, and personal. Its visual language should support one simple action at a time.

The desired feeling is:

- calm, but not sleepy
- premium, but not decorative
- human, but not sentimental
- minimal, but not empty
- alive, but never distracting

## Visual Position

Valwen should sit between wellness, clinical calm, and Nordic minimalism. It should avoid the typical cheap relaxation-app look: bright gradients, generic lotus imagery, cartoon clouds, playful illustrations, or motivational poster styling.

The interface should feel like a quiet object the user can return to during a difficult moment.

## Color System

### Night Theme

Night is the primary mood. It should feel deep, protective, and low-stimulation.

Main colors:

- Deep blue green: `#162a36`
- Muted teal: `#395f66`
- Soft moss teal: `#6f887c`
- Aqua light: `rgba(95,238,218,0.42)`
- Gentle green: `rgba(180,224,108,0.28)`
- Muted blue: `rgba(90,152,238,0.28)`
- Warm quiet light: `rgba(244,214,142,0.2)`

Use night theme as the reference for product quality. Other themes should feel like variants of this calm system, not separate designs.

### Day Theme

Day should be lighter, but not bright white and not sandy. It should keep darker edges and a calm center.

Main colors:

- Muted blue teal: `#54787e`
- Mist green: `#7fa095`
- Pale grey green: `#aebcae`
- Soft white mint: `rgba(238,252,247,0.34)`
- Pale green: `rgba(189,229,218,0.28)`
- Cool blue: `rgba(126,181,198,0.22)`

Day theme should always preserve enough edge depth for white text to remain readable.

## Texture

Grain is part of the identity. It should feel like film, water, or an old soft photograph, not like dirty noise.

Rules:

- Grain may be visible across the whole page.
- The central glow can have stronger grain than the edges.
- Grain should move subtly, like film or old TV texture.
- Avoid sharp grid artifacts, repeating tiles, or high-contrast speckles.
- Grain should calm the background, not compete with the text.

## Motion

Motion should feel slow, organic, and slightly behind the surface.

Principles:

- Use long easing curves.
- Prefer drifting, dissolving, breathing, and soft fading.
- Avoid bouncy, springy, playful, or gamified animation.
- Avoid fast side-to-side movement.
- Pointer reaction should be subtle and delayed, not direct.
- Button feedback should feel like a soft pulse, not a click effect.

Current motion direction:

- background light slowly wanders behind the text
- grain gently flickers
- card transitions use fade, slight blur, and small vertical movement
- button ripple expands slowly and disappears quietly

## Typography

Primary font stack:

`Inter`, `SF Pro Display`, `SF Pro Rounded`, `Segoe UI`, `system-ui`, `sans-serif`

Text should be large enough for stress moments, but not heavy or shouty.

Guidelines:

- Use medium weight, not bold.
- Keep letter spacing at `0`.
- Use generous line height.
- Keep lines short and centered.
- Avoid decorative fonts.
- Avoid all caps for emotional or instructional text.
- White text is part of the identity, but it needs soft shadow in light mode.

## Interface Shape

The interface should feel minimal and tactile.

Rules:

- The main card should not look like a normal card.
- Avoid visible boxes around the main instruction area.
- Buttons can be circular and soft, but should not look like toy buttons.
- Secondary controls should stay quiet.
- The menu should feel like a transparent overlay, not a settings page.
- Icons should be simple line icons.

## Logo And Identity

Current direction: use the name `Valwen` without a visible logo for now.

Rules for future logo work:

- Prefer a very simple mark over a detailed symbol.
- The mark should work as an app icon at small size.
- Avoid eye-like shapes, fingerprint complexity, medical crosses, hearts, lotus symbols, and generic wave logos.
- The wordmark can be enough if the visual system is strong.

## Do Not Do

Avoid:

- bright wellness gradients
- heavy glassmorphism
- purple-blue SaaS palette
- cartoon mental-health illustrations
- medical emergency red
- too many controls on the first screen
- explanatory text inside the main experience
- aggressive vibration or strong tap effects
- gamified progress or celebratory completion
- sharp geometric animation edges

## Product Feeling Checklist

Before accepting a visual change, ask:

- Does it help during a panic attack?
- Can a stressed user understand it immediately?
- Does it reduce pressure rather than add stimulation?
- Does it still look calm when viewed on a small phone?
- Does it feel premium without looking like marketing?
- Does it preserve the one-action-at-a-time principle?

## Next Visual Directions

Good next additions:

- a calm completion state after the final card
- a breathing mode without text
- refined app icon exploration
- simple README screenshots
- a tiny motion guide for future transitions

Avoid adding these too early:

- accounts
- complex settings
- statistics
- social sharing
- long onboarding
- too many breathing methods
