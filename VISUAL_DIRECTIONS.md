# Valwen Visual Directions

These are possible visual directions for Valwen. They are meant as creative options, not final decisions.

## 1. Film Calm

The current direction is closest to this.

Feeling:

- old soft film
- quiet light behind glass
- subtle grain
- slow movement
- gentle darkness around the edges

Best for:

- making the app feel premium and personal
- keeping the experience low-stimulation
- creating a recognizable atmosphere without adding icons or illustrations

Risk:

- too much grain can become visual noise
- too much darkness can feel heavy in day mode

Recommendation:

Keep this as the main direction for now. It already fits the project well.

## 2. Nordic Clinical Calm

Feeling:

- clean
- professional
- controlled
- slightly medical, but not hospital-like

Visual traits:

- cooler blue-green palette
- less grain
- more spacing
- sharper typography
- softer but simpler background motion

Best for:

- trust
- clarity
- mental-health credibility

Risk:

- can become too sterile
- may lose the emotional softness that makes Valwen feel personal

Recommendation:

Use small elements from this direction for menu/settings clarity, but not as the main atmosphere.

## 3. Water Surface

Feeling:

- slow ripples
- light under water
- soft pressure release
- organic motion

Visual traits:

- circular expanding movement
- pale aqua highlights
- darker edges
- more liquid animation
- less visible UI surface

Best for:

- breathing mode
- completion state
- background motion

Risk:

- if too literal, it can look like a relaxation screensaver
- ripples can become distracting during panic

Recommendation:

Use this for a future breathing mode, not necessarily for the whole app.

## 4. Quiet Object

Feeling:

- tactile
- minimal
- almost like a small physical device
- one button, one action

Visual traits:

- less background drama
- stronger central button
- very subtle shadows
- precise spacing
- calmer grain

Best for:

- making Valwen feel like a reliable tool
- reducing visual complexity

Risk:

- may become too plain
- can lose the current emotional richness

Recommendation:

Good direction if the app ever starts feeling too animated. Keep it as a balancing reference.

## 5. Soft Aurora

Feeling:

- northern light
- soft glow
- protective darkness
- slow emotional warmth

Visual traits:

- teal, blue, pale green, and faint warm light
- wider moving light fields
- blurred organic contours
- less visible interface

Best for:

- night theme
- premium emotional atmosphere
- strong first impression

Risk:

- can become too decorative
- can look like generic wellness if colors become too saturated

Recommendation:

Use carefully. The current animation already borrows from this direction; keep it restrained.

## Recommended Path

The best path for Valwen right now is a hybrid:

- base: Film Calm
- motion: Soft Aurora
- future breathing mode: Water Surface
- settings/menu clarity: Nordic Clinical Calm
- restraint check: Quiet Object

In simple words:

Valwen should feel like a quiet film-grain surface with soft moving light behind it. The interface should stay almost invisible, and every new feature should pass the question: does this help a person feel less pressure in a panic moment?

## Direction To Develop First

Develop Film Calm further.

It is already present in the current version, it fits the product, and it gives Valwen a more unique identity than a typical meditation or breathing app.

Next useful explorations:

- refine app icon in the Film Calm direction
- create GitHub/README screenshots using Film Calm
- design breathing mode using Water Surface, but keep the same grain and color system
- test a very subtle completion state using Soft Aurora light
